Dacha (Datengetriebener Chatbot) implementiert einen neuen Ansatz bei der Entwicklung eines Chatbots. 
Die Antwort auf eine Benutzerfrage wird auf der Basis der Ähnlichkeitsmaße in einer Datenbank mit Äußerungspaaren gesucht. 

Die Anwendung ist unter http://gaia.ift.cs.uni-potsdam.de/dacha/ abrufbar. 

Danke an https://github.com/romanleicht für die Unterstützung bei Entwicklung der Architektur für dacha, insbesondere für die
ursprüngliche Konfiguration von Maven, Spring, Hibernate, GWT.
